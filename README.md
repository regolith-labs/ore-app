# ORE App

Liquid digital gold. 

## Get Started

Install:
```sh
cargo install dioxus-cli
```
Build Tailwind:
```sh
npx tailwindcss -i ./input.css \
                -o ./public/tailwind.css \
                --config tailwind.config.js \
                --minify
```

Build App:
```sh
dx build
```

Serve:
```sh
dx serve --hot-reload
```

Visit http://localhost:8080/ to view the app.
